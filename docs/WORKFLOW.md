# Flujo de trabajo recomendado

## Crear una nueva mejora
```bash
git checkout -b feature/nombre-corto
```

## Guardar avances
```bash
git add .
git commit -m "feat: descripcion breve"
```

## Corregir un problema
```bash
git checkout -b fix/nombre-del-problema
```

## Integrar cambios
- Probar acceso
- Probar carga de Excel
- Probar mapa
- Probar Voronoi
- Probar exportaciones
- Actualizar `CHANGELOG.md`
